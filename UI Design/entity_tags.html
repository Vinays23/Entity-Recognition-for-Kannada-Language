Entity Recognition for Kannada Language


Tags Set



  • NN: Common Noun Name,Place,Thing

  • NST: NLoc Ex:ಪೂರ್ವ

  • PRP: Pronoun Ex:ನೀವು,ಅವರ

  • DEM: Demonstrative Ex:ಈ,ಆ

  • VM: Verb Ex:ಕೊಡಲಾಗಿದೆ

  • JJ: Adjective Ex:ಚಿಕ್ಕ,ಅನನ್ಯ

  • RP: Particles Ex:ಮಾತ್ರ

  • RB: Adverb Ex:ನಂತರ,ಹೀಗಾಗಿ

  • CC: Conjuction Ex:ಮತ್ತು,ಹಾಗೂ

  • WQ: Question Ex:ಏನು,ಹೇಗೆ

  • QF: Quantifiers Ex:ಕಡಿಮೆ,ಹೆಚ್ಚು

  • NUM: Numbers Ex:1,2,3,4

  • NEG: Negation Particles Ex:ಇಲ್ಲ

  • SYM: Symbol Ex:ಏ,ಎಲ್

  • PSP: Postpositions

  • COMP: Composition

  • UT: Quontative

  • ECH: Echo words

  • UNK: Unknown
